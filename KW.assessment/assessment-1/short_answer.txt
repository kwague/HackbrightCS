3) Explain the difference between these two commands in the terminal:
   
   $ cd Users
   $ cd /Users

Your answer:


`cd Users` is a relative path while `cd /Users` is an absolute path. For a relative path you are navigating to a subdirectory within the current directory, while for th absolute path you can navigate from anywhere in the file system

8) What is the difference between `git add` and `git commit`?

Your answer:


Git add is used to track files and any changes in them while got commit actually records changes to the repository. Git commit takes a current snapshot of the repository and permenantly saves a reference to it.
